# ml-docker
For building chihchiu29/ml-swiss-army-knife image.

The image is meant to be the all-in-one image for ML/DL. It contains commonly used DL tools and engineering tools.

# Included Tools

## ML/DL

* numpy / scipy / sympy / pandas / matplotlib
* TensorFlow
* scikit
* keras
* OpenCV, tesseract
* spacy

## Engineering tools

* git
* byobu
* emacs (nox)
